What is Js-Walker (Private MVC framework)

What New?

  • Highly and easy Confgurable
  • Configuration file : app.js
  • SSL Certificate
  • Database Switching (Mysql/Mongo)
  • Base path for resource.
  • Enable or disable socket handling
  • Error tracking for production and Development


  • Strong redis Support
  • Custom Node_modules for redis manipulation


  • Socket flavour
  • Single user(receiver) socket
  • Group socket
  • Self socket


  • Maintainability
  • Every Controller and module have strict approach


  • Tracking/Control Every Output log

    • Js-walker have unique feature that no framework has,Ajax and socket Response Tracking on development phase and easily disable in production phase

    Response from server can track from client side console and see which controller or model this response coming from with line number.



  • Error tracking

    • Normal / Ajax / Socket all kind of request can track error with file name and line number in minimal form and easily track error.

    This Framework track even error in faulty query string for databaseit was some difficult with normal framework because mysql query error return sql error object not error line.



  • Code generator

    • No headache of Controller/model/view/template generation

    Write this line to command line:

    node system/walker.js welcome


    Practice the basic example provided by default in Js-Walker

    This framework specially design for large scale product,so it is not very easy to handle and not for scaffolding application.

    See the example of controller and model it was little bit time consuming process first time,but it is concrete MVC structure.

    Once you understand the procedure for js-walker then it is Piece of cake for web engineer.
